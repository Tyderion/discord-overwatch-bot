# discord-overwatch-tyd
> A Bot for Overwatch Relate things: Stats, News etc

## Installation

Clone this repository, and run:
```sh
npm install
```

## Usage

Run Bot
```sh
npm start
```
Watch for changes and rerun 
```sh
npm run watch
```
